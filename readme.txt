Gamma
===================
Baubap AI Challenge 2.0
Clasifiación binaria
-----------
Archivos:

EDA.ipynb: notebook que explica cual fue la metodología usada para el preprocesamiento de los datos
Análisis.ipynb: notebook que explica como se obtuvieron los modelos

preprocessing.py: script con funciones que se utilizan para el preprocesamiento de los datos
evaluation.py: script que sirve para evaluar las predicciones de un modelo
models.py: script que contiene los modelos utilizados en el proyecto
fusion.py: script que contiene el algoritmo para fusionar las probabilidades de los modelos utilizados

training.csv: dataset de entrenamiento
data_evaluation.csv : datos de evaluación final
garbage.txt: contiene las features identficadas como artificiales
worst_features.pkl: contiene las features que no se tomaron en cuenta

gamma.mp4: video donde se explica el enfoque utilizado

Tambien viene un archivo pdf, el cual es un paper en el que me basé para las propuestas.

requirements.txt: contiene las librerías utilizadas de un entorno que ocupo para tareas de dl
		  las librerías más importantes son sklearn, tensorflow, keras, itertools y pandas.profiling.
